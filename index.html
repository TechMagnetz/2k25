Technical Events

Idea Sprint

Venue: Nila hall
  • Rule 1: Present your original technical paper within 5 minutes using PPT max of 10 slides
  • Rule 2: Exceeding time limit leads to point deduction.
  • Rule 3: Submit a detailed abstract during registration.

Innovibe Expo

Venue: Communication Lab
  • Rule 1: Maximum 4 people per team.
  • Rule 2: Each team will have 7 minutes to explain the project.
  • Rule 3: Software and Hardware-integrated software projects are accepted.
  • Rule 4: Participants should submit detailed abstract and 1 minute working model video during registration

Bug Bounty

Venue: IT Lab
  • Rule 1: Write code and debug errors with any programming languages.
  • Rule 2: Maximum 4 participants per team.
  • Rule 3: Plagarism leads to disqualification.

Web Wars

Venue: MBA Lab
  • Rule 1: 5 minutes to understand the topic given in spot.
  • Rule 2: 5 minustes of internet access to gather resources.
  • Rule 3: Use VS Code to build the webpage.
  • Rule 4: Complete the task within 30 minutes

Squid Game

Venue: Classroom-509
  • Maximum 3 participants per team.
  • Round 1- Technical Quiz: Online MCQ with 15 questions in 10 minutes.
  • Round 2- Action Pitch: On-spot problem, 10 minutes to brainstrom, 5 minutes to pitch.

Non-Technical Events

Adzap

Venue: Classroom-410
  • Rule 1: 1-4 members per team.
  • Rule 2: Time limit: 30-60 sec.
  • Rule 3: Advertisement can contain audio, video , skit. They can use resources to create a innovative ad.

Fire Crackers

Venue: Classroom-411
  • Rule 1: Each team must have a designated number of players- 2 for BGMI and 4 for Free Fire.
  • Rule 2: Players are required to use their official game accounts.
  • Rule 3: Participants must play on mobile phones or tablets.
  • Rule 4: Players caught using cheats or exploiting buys will face a ban

Shine Fest

Venue: Raman Auditorium
  • Rule 1: Solo and duo performance only allowed.
  • Rule 2: Time limit 3-6 minutes.
  • Rule 3: Performance can be any of your talent (Ex. Singing , dancing , Mimicry, Mime, Instrumental Play...)

View Finder

Venue: Nila Hall
  • Rule 1: Only individual entries allowed.
  • Rule 2: Photos unrelated to the theme will be disqualified.
  • Rule 3: No stock or AI generated images, it must be taken on the event day.
  • Rule 4: Photos must be in JPEG/PNG(1080p min) with brief description(50-100 words)

Pixel Bridge(Connection)

Venue: Classroom-409
  • Rule 1: Maximum 3 participants per team.
  • Rule 2: Rounds will be conducted as per the participants.
  • Rule 3: Each round has 10 questions, different and innovative tasks are conducted in this game.
